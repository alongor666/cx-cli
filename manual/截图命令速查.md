## 常见参数

| 命令 | 参数 | 说明 |
|------|------|------|
| `cx login` | `--token` | 直接传 PAT |
| `cx routes` | `--tag kpi` | 按标签过滤 |
| `cx query` | `--year=2026` | 查询参数 |
| `cx fields` | `--groupable` | 只显示可分组字段 |
| `cx metrics` | `--category ratio` | 按分类过滤 |
| 所有查询 | `-f json` | 切换输出格式 |

---

## 输出格式

- `table`：彩色表格（默认，命令行窗口）
- `json`：JSON 格式
- `csv`：CSV 格式

示例：`cx query KPI --year=2026 -f json`

---

## 故障排除

| 问题 | 解决 |
|------|------|
| `'cx' 不是内部或外部命令` | 检查 PATH，重启命令行窗口 |
| 登录失败 | 检查 PAT 格式和网络连接 |
| 无输出 | 可能是空结果或权限不足 |
| 中文乱码 | PowerShell 设置 UTF-8 编码 |
