## 故障排除

### 图片不显示
**问题**：浏览器显示占位符文字  
**解决**：
1. 检查文件名是否完全匹配（区分大小写）
2. 检查图片是否在 `images/` 目录下
3. 按 `Ctrl+F5` 强制刷新浏览器
4. 检查浏览器控制台是否有 404 错误

### 图片模糊
**问题**：截图文字不清晰  
**解决**：
1. 使用 PNG 格式（不要用 JPG）
2. 提高截图分辨率（1920×1080 或更高）
3. 命令行窗口字体调大一些

### 命令执行失败
**问题**：`'cx' 不是内部或外部命令`  
**解决**：
1. 确认 `C:\tools\cx\` 已添加到 PATH
2. **重启所有命令行窗口**
3. 运行 `cx --version` 测试
4. 检查 `cx-windows.exe` 是否在 `C:\tools\cx\` 目录
